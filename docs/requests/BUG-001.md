# BUG-001
## Sources returned in chat response don't match the response.
**Description:** Sources returned in chat response don't match the response.  There are sources listed that were not referenced in the source.  She shouldn't have either because they are outside the filtering criteria.
**Steps to Reproduce:** Enter into the Message box "List out job search activities from March 16 onwards.
**Expected Behavior:** 
- Any notes referenced in the response should show up.
- All notes mentioned as a source should also have been referenced in the response.

## Chat pane text not selectable
**Description:** Text from both the user and assistant cannot be selected for copying.
**Steps to Reproduce:** 
- Enter any chat message
- Attempt to select the text
**Expected Behavior:**
- Text should be selected

## Vault organization prompt not being honored
**Description:** When asking the model to report entries `over the last 2 weeks`, the model responds with `Insufficient evidence.  Could you please narrow down your question by specifying a folder or tag, such as "daily/jobhunt/application" or "2024-04", to help me locate relevant notes?
**Steps to Reproduce:** 
- Use as the Vault Organization Prompt:
  > Daily notes live in `daily/**/YYYY-MM-DD.md`.  Each day I use these following headings when relevant:
  > `# Job Search` or `Job Hunt` - Activities and notes regarding my job search.   Tagged `#jobhunt`.  Job applications are tagged with `#jobhunt/application` and application responses are `jobhunt/application/response`.
  > Time-based questions ("last two weeks", "this month") should filter daily notes by filename date.
  
- Enter `List out my job search activities over the last 2 weeks`
**Expected Behavior**: The model should return notes filtered by title under any folder within `daily`

## Punctuation causes exception
**Description:** Entering a chat request with `?`, `!`, `` ` `` or `.` will return an exception slideout saying `` Chat failed: fts syntax error near `?` ``
**Steps to Reproduce:** 
- Enter something with `?` in the user prompt.
**Expected Behavior:**
- The prompt should treat that sentence as a question.
- It should be possible to terminate a sentence with punctuation.
- It should be possible to use `` ` `` in a prompt.
